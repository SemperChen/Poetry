## Running
#### Android:
1. run `npm i`.
2. run `react-native run-android`.
#### IOS:
1. run `npm i`.
2. run `sudo gem install cocoapods`.
3. run `cd ios&&pod install --repo-update`.
4. run `react-native run-ios`.
